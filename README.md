# Bright Data

Bright Data is an all-in-one web data platform providing proxy networks, scraping browser automation, web scraper APIs, and dataset delivery services for large-scale web data collection and aggregation. Serving eCommerce, financial services, real estate, ad tech, and AI/ML industries.

## Products

- **Web Scraper API** - Pre-built and custom scraping endpoints with structured data extraction, proxy rotation, JavaScript rendering, and CAPTCHA solving
- **Scraping Browser API** - Fully managed cloud browsers for Selenium, Puppeteer, and Playwright workflows
- **SERP API** - Multi-engine search results delivered on-demand
- **Browser API** - Remote browser infrastructure with stealth capabilities
- **Proxy Networks** - 400M+ residential IPs, 1.3M+ ISP IPs, and datacenter proxies across 195 countries
- **Dataset Marketplace** - 5B+ regularly refreshed records across industries
- **Web Archive** - Petabyte-scale historical web data
